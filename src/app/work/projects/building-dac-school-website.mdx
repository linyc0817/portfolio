## Overview

Developed the frontend of the department website using Vue.js, and customized the WordPress backend with Gutenberg to meet client-specific needs.

## Key Features

- Built and maintained the Department of Applied Chemistry website (dac.nycu.edu.tw), **serving 3~4K monthly visitors** and **~20GB monthly traffic** (SimilarWeb & AWS CloudFront analytics).
- Delivered responsive, accessible UI with Vue and Tailwind CSS, ensuring seamless display across
devices.
- Implemented Bitbucket CD pipeline for automated deployment to AWS EC2 upon PR merges,minimizing manual errors.
- Created custom WordPress Gutenberg blocks, enabling non-technical staff to manage content efficiently.

## Technologies Used

- **Vue.js**: For flexible, customizable user interface.
- **Zeplin**: For creating and managing design assets and prototypes.
- **Tailwind CSS**: For styling Vue components with a modular, themable approach.
- **Wordpress**: For no technical background assistant to easy maintain website content.

## Outcome

This product is still using as the website of [Department of Applied Cheminstry](https://www.dac.nycu.edu.tw/?lang=zh) in National Yang Ming Chiao Tung University.
