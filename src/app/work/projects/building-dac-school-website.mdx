## Overview

Developed the frontend of the department website using Vue.js, and customized the WordPress backend with Gutenberg to meet client-specific needs.

## Technologies Used

- **Vue.js**: For flexible, customizable user interface.
- **Zeplin**: For creating and managing design assets and prototypes.
- **Tailwind CSS**: For styling Vue components with a modular, themable approach.
- **Wordpress**: For no technical background assistant to easy maintain website content.

## Outcome

This product is still using as the website of Department of Applied Cheminstry in National Yang Ming Chiao Tung University.
